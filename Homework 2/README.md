# Homework 2 – Material UI "Team Profile App"

A Flutter application introducing **COIL Flutter - Group 5**, a collaborative project between TAMK (Finland) and THWS (Germany) as part of the Cross Platform Development course.

---

## About the App

This app presents our team and all team members in an interactive and visually appealing way using **Material UI** widgets.

---

## Features

- Team overview with name and description
- Swipeable member cards (left/right navigation)
- Tap any member card to view full profile details
- Hero animation between main card and detail page
- Member details: name, country, role, hobbies, and personal motto
- Profile photos for team members
- Page indicator dots
- Built with Material 3
